# Module 12: Mastering JavaScript Objects

Welcome to Module 12! This module covers everything you need to know about JavaScript Objects, from the absolute basics to advanced topics like shallow/deep copies, object immutability, destructuring, and modern optional chaining.

## 📁 Files & Topics Covered

Here is the index of scripts and guides in this folder:

1. **[01-objects-basics.js](01-objects-basics.js)**: Introduction to object declarations, property access (dot vs bracket notation), and standard usage.
2. **[02-constructor-functions.js](02-constructor-functions.js)**: Creating multiple object instances using Constructor Functions and the `this` keyword.
3. **[03-factory-functions.js](03-factory-functions.js)**: Generating object instances dynamically using Factory Functions.
4. **[04-in-and-for-in.js](04-in-and-for-in.js)**: Checking if property keys exist using the `in` operator and looping properties with `for...in`.
5. **[05-object-keys.js](05-object-keys.js)**: Working with built-in utility arrays using `Object.keys()`, `Object.values()`, and `Object.entries()`.
6. **[06-object-references.js](06-object-references.js)**: Understanding how objects are stored as references in memory instead of direct values.
7. **[07-static-methods.md](07-static-methods.md)**: Conceptual guide on static vs instance methods using `Object.assign()` as an example.
8. **[08-shallow-vs-deep-copy.md](08-shallow-vs-deep-copy.md)**: Deep dive into the difference between shallow copying (references copied) and deep copying (cloning nested values).
9. **[09-seal-freeze-has-own.md](09-seal-freeze-has-own.md)**: Guide on object immutability with `Object.seal()`, `Object.freeze()`, and checking owned properties with `Object.hasOwn()`.
10. **[10-object-destructuring.js](10-object-destructuring.js)**: Unpacking property values from objects into distinct variables using modern syntax.
11. **[11-dynamic-object-creation.js](11-dynamic-object-creation.js)**: Creating objects dynamically based on runtime user inputs.
12. **[12-optional-chaining.js](12-optional-chaining.js)**: Safely navigating deeply nested properties without crashing using the `?.` operator.

## 🚀 How to Run the Examples
You can run any of the JavaScript scripts in this module directly using Node.js:

```bash
node 01-objects-basics.js
node 02-constructor-functions.js
```
